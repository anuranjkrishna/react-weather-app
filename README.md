# 🌦️ React Weather App

## 📌 Project Description
The React Weather App is a responsive web application that provides real-time weather information for different cities using the OpenWeather API.  
Users can search for any city and view current weather conditions with a clean and user-friendly interface.

---

## 🚀 Features
- Search Weather by City Name
- Real-Time Weather Data
- Dynamic Weather Icons
- Loading Spinner
- Error Handling
- Responsive Design
- User-Friendly Interface

---

## 🛠️ Technologies Used
- React JS
- JavaScript
- HTML
- CSS
- OpenWeather API
